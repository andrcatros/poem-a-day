About

This website updates with a new poem every day using the Poems One API. I created this website using just Vanilla Javascript, HTML and CSS. You can check out my code in the GitHub repository for this website here.

I plan to create my own poetry API in order to be able to add more features (including: author biographical information, publication date and a random poem page). However, this might take a while.
